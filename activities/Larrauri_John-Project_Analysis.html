Project Analysis

Client: Top Stone, Inc.

Project: Website Redesign

Designer: John Larrauri

Site Intent

The objective of this project is to find the best way to redesign the website www.topstone1.com to help increase site visits by making the site more user friendly, visually appealing, and increase overall business.

During the analysis, there are several issues with the site. The site is visually unappealing, there is no clear navigation elements anywhere, no company branding, ie: logo, and nothing that makes a site visitor know what the site is for. Navigating the site is very difficult and even with use of the site map, the site is entirely too difficult to navigate. The setup of the site is a total mess. There is no clear CTA and no way at all for a user to know what page they are visiting at any particular moment. There is no consistency amongst the pages and consistency is even lost from one part of the current page to another part of it. The visually weak and inconsistent design allow for me to have a lot of room to make overall improvements on the site including, but not limited to, a more modern design which will create a better user experience to make site visitors want to stay on the site and potentially increase the overall business of the physical store. W3C validation in Firefox shows 98 errors in the html code and 1 error in the CSS code. There is a lot of room for overall improvement of the site.

For a better overall user experience and a higher conversion rate, my redesign will include, but not be limited to, the following:

  1. Website content needs to be organized into 6 pages including a home and form page
  2. Add escape hatch logo & clear cut navigation elements to the head of all pages
  3. Include a clear cut CTA on the home page
  4. Decrease the overall number of images and organize them better
  5. Look into implementing a constantly scrolling gallery of images on the home page
  6. Promote the physical location of the store to help turn site visitors into walk-in customers
